##2.两数相加(Python3)

给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。

如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。

您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

这道题本来想用链表转换成整数相加再换回链表的，但考虑到这样做比较无脑，于是放弃了该方法。中规中矩的利用了去年数字逻辑学的全加器的思想，设置进位标志，逐个相加，其实这样做也很无脑，但限于个人水平有限，只能想出这种方法，代码无非是if else的简单组合，但是要考虑的情况比较多。比如在加完后如果溢出位为1还要再在结果后加一位。
